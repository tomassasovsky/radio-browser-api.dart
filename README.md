## Installation ðŸ’»

This package requires [Dart][dart_install_link] 2.12.0 or later.

Add `radio_browser_api` to your `pubspec.yaml`:

```yaml
dependencies:
  radio_browser_api:
```

Install it:

```sh
dart pub get
```
